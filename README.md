# 🛡️ ShieldCheck — Advanced Password Security Analyzer

> **Analyze. Strengthen. Secure.**
> A modern cyber-themed password security analyzer that evaluates password strength in real time using interactive visualizations, security metrics, and intelligent recommendations.

---

## 📖 Overview

**ShieldCheck** is a responsive web application that helps users understand how secure their passwords are. Instead of simply displaying a strength label, the application performs multiple security checks, calculates entropy, estimates brute-force crack time, visualizes character composition, and provides recommendations for creating stronger passwords.

Designed with a futuristic cybersecurity-inspired interface, ShieldCheck combines usability with engaging visual feedback to make password security easier to understand.

---

## ✨ Features

### 🔐 Real-Time Password Analysis

* Instant password strength evaluation
* Live security percentage meter
* Dynamic security level indicator
* Color-coded strength visualization

### ✅ Password Validation Checklist

Automatically checks whether the password contains:

* ✔ 12+ Characters
* ✔ Uppercase Letters
* ✔ Lowercase Letters
* ✔ Numbers
* ✔ Special Symbols

### 📊 Interactive Security Dashboard

#### Character Composition Chart

Visualizes the distribution of:

* Uppercase characters
* Lowercase characters
* Numbers
* Special symbols

#### Security Radar Chart

Displays multiple security vectors including:

* Password Length
* Character Variety
* Uniqueness
* Complexity
* Entropy Score

### 🧠 Security Intelligence

The analyzer provides:

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/ShieldCheck.git
```

2. Navigate into the project folder

```bash
cd ShieldCheck
```

3. Open `index.html` in your preferred browser.

No installation or build process is required.

## 🔍 Password Analysis Criteria

The application evaluates passwords based on multiple security factors:

* Password length
* Character diversity
* Character uniqueness
* Entropy calculation
* Complexity score
* Dictionary password detection

The final strength score is calculated using these combined security metrics.
